# Favicon Creation Instructions

## Create Favicon from Your Logo

To create a favicon from your Swaad-E-Sehat logo:

1. **Save your logo as favicon files:**

   - `favicon.ico` (32x32 pixels) - Main favicon
   - `favicon-16x16.png` (16x16 pixels)
   - `favicon-32x32.png` (32x32 pixels)
   - `apple-touch-icon.png` (180x180 pixels)

2. **Place files in:** `public/` directory

3. **Recommended tools:**
   - Online favicon generators
   - Image editing software (Photoshop, GIMP)
   - Favicon.io website

## Your Logo Features

- **Circular Design**: Perfect for favicon
- **Dark Green Circle**: Will work well at small sizes
- **High Contrast**: Text and elements visible when scaled down

## Files Needed

```
public/
├── favicon.ico
├── favicon-16x16.png
├── favicon-32x32.png
└── apple-touch-icon.png
```

Once you create these files, the website will automatically use them as browser tab icons!
